# WP-Patch

Wordpress by Version
--------------------

Just update Wordpress by Version.

Update on release as soon as possible.

## Install

```bash
$ git clone https://github.com/b2b-web-id/wp-patch
$ cd wp-patch
$ git submodule update --init
```
